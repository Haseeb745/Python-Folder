001 : IoT based Plant Habitat

Using Raspberry Pi 3 kit having Raspbian OS and Python language with MYSQL Database and ThingSpeak webserver to store our data so as to access it from anywhere around the world.

Plant watering is the most important practice and most labour intensive work of today’s contemporaneous task life thereby leading to automation of things periodically. No matter which ever weather it is hot, dry, cloudy or wet, we need to know to water the plants regularly with the regulated flow of water. But this manual process is very hectic which involves two activities i.e when to water and how much to water the plant. Both of these manual activities are taken care by the proposed system.

This project replaces the Arduino microcontroller by Raspeberry Pi which acts as control block of the complete system. This Raspberry Pi uses the Firmata Protocol so that it communicates with all the components of the system. The main aim of this project is to modernize and automate the watering system so that plants never go thirsty again. The project is to develop an IoT based system that helps one to maintain the garden even in their absence by watering the plants as and when needed. The components which are integrated analyze the need to water the plant and signal the system buzzing an alarm, and switches on the motor automatically. Then the system itself handles the watering of plant with regulated water flow analyzing the moisture present in the soil helping to reduce wastage of water. The system data automatically gets updated and is permanently stored in the azure cloud. This system when once installed and turned on needs no regular manual effort to water the system.
