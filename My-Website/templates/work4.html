004 : Password Health Checker

A program that determines the health of the password by checking how many times it has been hacked.

Here I made use of an API of a website named Have I Been Pwned to check the health of our password. The password here is passed to the program which connects to the API and provide us the result.

With the help of requests module(a libraray in Python) I connected to the API of the above website. I also made use of hashlib library in here to hash our password as I have to provide the password to the website which may store our original password if not hashed. I then requested the API for all the hashed passwords from its database that match the hashed password and then provide the count for how many times it was hacked and suggest as to wheteher use to it or not.
