002 : My Perosnal Website

Using Python programming along with HTML/CSS and JavaScript files.

This was my very first personal website which I have developed from scratch and published online using pythonanywhere, an online IDE.

In this project I have made use of FLASK framework of Python to develop the WebServer. Using multiple methods of this framework in my server.py file along side using the HTML/CSS file for my web content and styling methods and Java Script file to add extra content to website, I developed this website fully on my knowledge of those programming languages and reading its content as and when needed. Later on by cloning this content to the pythonanywhere (which is an online IDE and Web Hosting service based on Python and provides in-browser access to server-based Python and Bash command-line interfaces, along with a code editor with syntax highlighting) server I have successfully hosted this website onto public server.
