006 : Web Scraping

Scraped the data from a website successfully based on the HTML content of the page.

Web Scraping means to grab the data or get hold of the data from a website programatically i.e. download a web page scrape the data that we want from the website and clean the data by keeping only the information of our interest, for example to get all the jobs posted by a website.

Working with multiple packages and libraries provided by Python in this project, I have connected to a website which offers its data to be scraped for different purposes. Bot like Google Bot is what Google uses to index the website which means it is a crawler program to understand what website is all about and then pickups the website according to its content and pushes to Database. To check for what we can and can't scrape on a website we need to check for the 'robots.txt' file of website and look out for its content. Now here in this project I have created a program to scrape website based on its HTML tags and classes and get only that data from the website which is of interest to me like the web content which has more than a 100 likes or votes. To have this done we need to get hold of the HTML content of the page and inspect its elements then convert this HTML to text using BeautifulSoup provided by python and then scrape the needful content out of it.
